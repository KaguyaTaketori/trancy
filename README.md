# Trancy

A Telegram bot for translation and language learning.

## Features

- Translation between multiple languages
- Language learning utilities
- Search functionality

## Setup

```bash
pip install -r requirements.txt
```

## Configuration

Edit `.env` and `config.json` with your API credentials.

## Usage

```bash
python bot.py
```
